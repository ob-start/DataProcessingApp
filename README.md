# DataProcessingApp
本程序是由 Python 编写的数据处理工具，经过 PyInstaller 打包为独立的 Windows 可执行文件（.exe），可在无 Python 环境的计算机上直接运行。

操作系统：Windows 7 / 8 / 10 / 11（32位或64位）

运行权限：建议以普通用户权限运行，如涉及文件读写请确保路径可访问

无需安装 Python：所有依赖均已打包进 .exe 文件中
# 功能完整说明：
✅ 支持手动输入数据或选择文件绘图；
✅ 支持识别极大值和极小值；
✅ 支持根据输入的阈值筛选极大值/极小值；
✅ 支持放大功能（鼠标左键拖动）；
✅ 支持重置视图；
✅ 支持导出图表为图片（PNG/JPG/SVG/PDF）；
✅ 支持导出数据到 Excel；
